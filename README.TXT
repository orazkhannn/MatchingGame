PROJECT TITLE: MatchingGame
PURPOSE OF PROJECT: The purpose of this project is to create a digital version of the classic memory-matching card game. Players must flip over pairs of cards to find matching values. The game challenges the player's memory and provides a functional example of object-oriented programming in Java, specifically utilizing 2D arrays and basic GUI interaction through the comp102x library.
